1000 Years Validator

MML インスタンスのバリデーション・チェックを行うことができます。1000年カルテプロジェクトへ参加するためには、医療情報システムからMML形式のデータを出力する必要があります。このサーバーでその際の検証を行うことができます。

検証結果について

  • 検証は MML4.0 の XSD に対して行います。
  • 結果は JSON オブジェクトで表示されます。
  • result: "valid" であれば妥当なインスタンスです。
  • result: "NOT valid" の時は問題があります。理由は reason 属性に表示されます。
  • 検証プログラムは Java ネイティブの javax.xml.validation パッケージを使用しています。
